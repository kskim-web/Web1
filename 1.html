HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

- 기본적인 HTML Tag -

Strong Tag: <strong> </strong>
Tag 사이의 글자를 진하게 표시

Underline Tag: <u> </u>
Tag 사이의 글자에 밑줄 표시

Heading Tag: <h1>~<h6> </h1>~</h6>
제목으로 지정 / 검색의 기준이 됨.

Line Breaker Tag: <br>
줄바꿈

Paragraph Tag: <p> </p>
단락, 예) <p style="margin-top:45px;"> 단락 위에 45px만큼 띄움

Image Tag: <img>
이미지 삽입, 예시: <img src="coding.jpg" width="100%"> src는 Source의 약자로 이미지파일 경로, width는 사진의 크기를 조정, %는 화면에 따라 자동 조절

List Tag: <li> </li>
목록, Unordered List: 숫자없음 <ul> </ul> / Ordered List: 숫자있음 <ol> </ol>

Title Tag: <title> </title>
제목, 검색할 때 가장 중요

Meta Tag: <meta charset="utf-8">
문자 인코딩, 웹페이지가 UTF-8로 만들어졌으니 웹브라우저에서 UTF-8로 열도록 설정

Head Tag: <head> </head>
본문을 설명, body - /body Tag: 본문 으로 구분하여 사용

Doctype Tag: <!doctype html>
HTML로 만들어졌다는것을 표현, 문서의 시작 부분에 추가

HTML Tag: <html> </html>
head Tag와 body Tag를 감싸는 tag

Anchor Tag: <a> </a>
링크삽입, 예시: <a href="링크할 파일이나 사이트 주소" target="_blank" title="Tooltip">
Target="_blank": Link를 여는 방법 = 새로운 Tab으로 열림, title: 마우스 커서를 올리면 Link에 대한 설명 표시
